# 题库索引（Problems）

本目录用于存放洛谷题目（或其它在线题目）的完整题目、解析与 Python 解答。每道题放在单独子目录下，目录内通常包含：

- README.md：题目描述（或题目摘要 + 链接）
- analysis.md：详细解析（思路、伪代码、复杂度分析、注意事项）
- solution.py：可运行的 Python 解答（含注释、输入输出说明）
- tests/：若干测试用例或 pytest 测试文件

点击下面的链接可以跳转到题目页面（在 GitHub 上打开子目录的 README）：

- [两数之和（示例题）](./problems/two_sum_两数之和/README.md)
